# ADR-V6-004: 产品定位——数据资产引擎为第一理由（REV-7 决策③）

- **Status**: Accepted
- **Date**: 2026-07-17
- **Refs**: V6 架构设计 §0.6 决策③ / §0.5 护城河 / §1 北极星

## Context
用户来装这个软件的「第一理由」定位。产品目标不变（获取个人数据资产），操作电脑是 V6 电脑端形态下的新增能力。

## Decision
- **第一理由 = 个人数据资产引擎**（人生图谱 / PTG / HL-12），是产品本体，占 ~70% 新建工作量。
- **操作电脑 = 新增能力 + 数据捕获场**：身体全量继承（白送），Phase 1 先原样跑；主战场放在大脑（捕获 → 原子化 → PTG → 洞察 → Personal RAG 飞轮）。
- **卖点叙事**遵 §0.5④「数据是使用副产品」铁律：对外文案只讲「记住 / 帮你 / 越用越懂 / 你的」，**永不出现**「采集 / 上传 / 监控 / 抓取」。

## Consequences
- ✅ Phase 1 主战场明确 = brain；身体不占用新建精力。
- ✅ 数据飞轮：用 agent 操作电脑干活 → 过程数据沉淀 PTG → agent 越懂用户 → 越离不开（§0.5 护城河① 数据壁垒越用越厚）。
- ✅ 与 §5.1 硬约束（只记流经自身的数据）互为表里，合规话术与心理安全统一。
